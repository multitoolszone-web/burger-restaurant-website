# 🍔 Burger Blast - Responsive Restaurant Website

## Project Name
**Burger Blast** - A modern, fully responsive restaurant website for a burger joint.

## Description
Burger Blast is a complete restaurant website designed to provide an immersive experience for food lovers. It features a vibrant red, yellow, and black color scheme, high-quality images, and smooth animations. The website includes all essential pages for a restaurant business: Home, Menu, About, Contact, and a fully functional Cart/Order page with localStorage integration.

## Features
- ✅ **Fully Responsive Design** - Works seamlessly on mobile, tablet, and desktop
- ✅ **Sticky Navigation Bar** - Easy access to all pages while scrolling
- ✅ **Interactive Menu** - Filter by categories (Veg, Chicken, Combos, Drinks)
- ✅ **Shopping Cart System** - Add/remove items, update quantities, calculate total
- ✅ **localStorage Integration** - Cart data persists across pages
- ✅ **Form Validation** - Contact form and checkout form validation
- ✅ **Offer Popup** - Special discount notification on first visit
- ✅ **WhatsApp Integration** - Floating button for quick orders
- ✅ **Smooth Scrolling & Animations** - Enhanced user experience
- ✅ **Hover Effects** - Interactive elements with visual feedback
- ✅ **Google Maps Integration** - Location map on contact page
- ✅ **Testimonials Section** - Customer reviews showcase
- ✅ **Image Gallery** - Instagram-style photo gallery

## Pages List
1. **Homepage (index.html)** - Hero section, featured burgers, why choose us, combo offers, about preview, testimonials, gallery, CTA section
2. **Menu Page (menu.html)** - Categorized menu items with add to cart functionality
3. **About Page (about.html)** - Story, mission, vision, and team member profiles
4. **Contact Page (contact.html)** - Contact form, Google map, and business details
5. **Cart/Order Page (cart.html)** - Shopping cart, quantity management, checkout form

## Technology Stack
- **HTML5** - Structure and content
- **CSS3** - Styling, animations, responsive design
- **JavaScript (ES6)** - Interactivity, cart system, form validation
- **Font Awesome** - Icons and visual elements
- **Google Fonts** - Typography
- **localStorage** - Persistent cart data
